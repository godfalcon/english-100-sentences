# 100个句子记完1200个小学单词 - 在线学习网站

基于俞敏洪《100个句子记完1200个小学单词》制作的在线学习工具。

## 功能特点

- **18个主题分类**：按主题浏览100个英语句子
- **句子音频播放**：点击播放按钮听句子朗读
- **单词双击发音**：双击句子中的任意单词即可听发音
- **词汇卡片**：展示核心词汇和主题词汇，点击可查看详情并发音
- **语法笔记**：可展开查看每个句子的语法要点
- **响应式设计**：支持PC和移动端访问

## 使用方法

### 本地运行

1. 进入项目目录
2. 启动本地服务器：
   ```bash
   python3 -m http.server 8080
   ```
3. 打开浏览器访问：http://localhost:8080

### 在线部署

可直接部署到任意静态托管服务，如：
- GitHub Pages
- Vercel
- Netlify
- 云开发静态托管

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- 单词发音：有道词典 API
- 句子音频：本地 MP3 文件
